# Session Summary (vNext)

## Goals

- 结算不“审判”，而是：
  - 反馈事实（时长、分段）
  - 轻量鼓励（与数据一致）
  - 可选的情绪记录（1 次点击完成）

## Key UX

- 默认保持 **一屏可读**：重要信息优先，分段列表可滚动。
- “情绪记录”必须 **可跳过**，且不增加键盘输入负担（优先点击）。
- 提供 **导出小卡**，用于分享/保存本次总结。

## Copy

- Focus 多：`“你保持了清晰的节奏。”`
- Break 多：`“你也在照顾自己——休息是计划的一部分。”`
- 很短会话：不弹结算（已在逻辑中处理）。
